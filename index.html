DeiCalc

The Calculator you want to use Everyday

DeiCalc is a good-looking, powerful and easy-to-use calculator designed for users of all levels. It supports basic calculations and advanced scientific functions. Featuring in a real-time updated result, easy-to-read expression view, powerful history tape and much more!

Optimized Portrait/Landscape Display

You are able to access all basic operations and advanced scientific functions in both portrait and landscape modes.

Result

DeiCalc updates the result automatically as you type. You don't need to move your finger across the huge screen and pressed the equal sign to get the final result.

Constants

DeiCalc provides you with instant access to common constants, such as pi and euler's number. You will be able to add custom constants in future versions
